# Online Job Portal System
An internship project — a web-based job portal built using PHP, CSS, and MySQL.

## Features
- Employer registration and job posting
- Job Seeker registration and job search
- Admin panel for management
- Contact and About pages

## Tech Stack
- Frontend: HTML, CSS
- Backend: PHP
- Database: MySQL (XAMPP)

## How to Run
1. Install XAMPP
2. Clone this repo into the htdocs folder
3. Import the database from the Connections folder
4. Start Apache & MySQL in XAMPP
5. Open localhost/Internship-Project in browser
